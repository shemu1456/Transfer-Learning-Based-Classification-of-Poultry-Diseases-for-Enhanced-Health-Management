Project Description

The PoultryDetect project is an innovative web application designed to assist poultry farmers and enthusiasts in the early detection of common poultry diseases. Utilizing state-of-the-art deep learning models, the application provides an intuitive platform for users to upload images of affected poultry or their droppings, which are then analyzed to identify potential infections.

Our core technology leverages a pre-trained Convolutional Neural Network (CNN) model (`healthy_vs_rotten.h5`) that has been fine-tuned on a diverse dataset of healthy and diseased poultry images. The Flask framework serves as the backend, handling image uploads, model inference, and rendering dynamic HTML pages to present the results. This project aims to contribute to better poultry health management, reducing losses and promoting sustainable farming practices.
